# Rubrica Python

Semplice applicazione da terminale per gestire una rubrica contatti, sviluppata in Python.

## Funzionalità

* Aggiungere un contatto (nome + numero)
* Visualizzare tutti i contatti (ordinati alfabeticamente)
* Cercare un contatto
* Eliminare un contatto
* Salvataggio automatico su file JSON

## Tecnologie utilizzate

* Python
* JSON 

## Come eseguire il progetto

1. Clona il repository:

```
git clone https://github.com/ErikaCensi/Rubrica_python
```

2. Entra nella cartella:

```
cd rubrica_python
```

3. Avvia il programma:

```
python rubrica.py
```

## Struttura del progetto

* `python rubrica.py` → logica principale
* `rubrica.json` → file dati
